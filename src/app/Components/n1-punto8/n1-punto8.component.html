Present Simple

How to use the present simple

The present simple is usually used to talk about one of two things:

  1. Habits.
  2. Things that are true all the time or that we see as permanent.

Note that the present simple is not used to talk about what is happening now.

Structure of the present simple to form sentences

This verb tense has a basic structure for its affirmative, negative and interrogative modes.
In the following table you can get to know it better.

Afirmativa Sujeto + verbo + complemento
Negativa Sujeto + auxiliar do/does not + verbo + complemento
Interrogativa Auxiliar do/does + sujeto + verbo + complemento
